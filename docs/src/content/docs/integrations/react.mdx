✨ Use react eslint config preset in witheslint. ✨

<Aside type="tip">
  This preset is included in the `witheslint` package
</Aside>

### Installation

<Tabs>
  <TabItem label="pnpm">
  ```sh
  pnpm add -D eslint witheslint
  # pnpm add -D eslint @witheslint/core @witheslint/preset-react
  ```
  </TabItem>
  <TabItem label="yarn">
  ```sh
  yarn add -D eslint witheslint
  # yarn add -D eslint @witheslint/core @witheslint/preset-react
  ```
  </TabItem>
  <TabItem label="npm">
  ```sh
  npm install -D eslint witheslint
  # npm install -D eslint @witheslint/core @witheslint/preset-react
  ```
  </TabItem>
</Tabs>

### Reference

- `eslint.config.js`

```js
import { defineConfig, presetReact } from 'witheslint'
// import { defineConfig } from '@witheslint/core'
// import { presetReact } from '@witheslint/preset-react'

export default defineConfig({
  presets: [
    presetReact(),
  ],
})
```
