Use WithESLint to make managing ESLint configurations for your project easier. This guide will assist you in getting started with your project.

<Aside type="danger" title="Requirements">
  - ESLint v9.10.0 and above
  - Node.js v18.18.x and above
</Aside>

### Installation

<Tabs>
  <TabItem label="pnpm">
  ```sh
  pnpm add -D eslint witheslint
  ```
  </TabItem>
  <TabItem label="yarn">
  ```sh
  yarn add -D eslint witheslint
  ```
  </TabItem>
  <TabItem label="npm">
  ```sh
  npm install -D eslint witheslint
  ```
  </TabItem>
</Tabs>

### Configuration

- Create ESLint configuration file `eslint.config.js`

<Tabs>
  <TabItem label="ESModule">
  ```js
  // eslint.config.js
  import { defineConfig } from 'witheslint'

  export default defineConfig()
  ```
  </TabItem>
  <TabItem label="CommonJS">
  ```js
  // eslint.config.js
  module.exports = (async () => {
    const { defineConfig } = await import('witheslint')
    return defineConfig()
  })()
  ```
  </TabItem>
</Tabs>


> Note that `.eslintignore` no longer works in Flat config, see [customization](/guides/customization) for more details.
