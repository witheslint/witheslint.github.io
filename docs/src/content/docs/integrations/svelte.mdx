✨ Use svelte eslint config preset in witheslint. ✨

### Installation

<Tabs>
  <TabItem label="pnpm">
  ```sh
  pnpm add -D eslint @witheslint/core @witheslint/preset-svelte
  ```
  </TabItem>
  <TabItem label="yarn">
  ```sh
  yarn add -D eslint @witheslint/core @witheslint/preset-svelte
  ```
  </TabItem>
  <TabItem label="npm">
  ```sh
  npm install -D eslint @witheslint/core @witheslint/preset-svelte
  ```
  </TabItem>
</Tabs>

### Reference

- `eslint.config.js`

```js
import { defineConfig } from '@witheslint/core'
import { presetSvelte } from '@witheslint/preset-svelte'

export default defineConfig({
  presets: [
    presetSvelte(),
  ],
})
```
